# Guia do Usuário - HUB BI

Esta plataforma foi desenhada para oferecer insights rápidos e monitoramento de missão crítica com uma interface premium e intuitiva.

## 1. Dashboard Principal
O coração do sistema. Aqui você encontra:
- **KPI Cards**: Indicadores de performance com gráficos de tendência (sparklines).
- **Gráficos Dinâmicos**: Visualizações de vendas, faturamento e metas que se atualizam em tempo real.
- **Alertas**: Notificações visuais no canto superior direito sobre eventos importantes.

## 2. Monitor NOC (Modo Noturno/TV)
Projetado para exibição em telões e centros de operação.
- **Visualização Full-Screen**: Otimizada para resolução 1080p.
- **Status do Sistema**: Monitoramento de uptime e latência do banco de dados.
- **Atualização Automática**: Os dados são renovados periodicamente sem necessidade de recarregar a página.

## 3. Gestão de Scripts e Layouts
(Funcionalidade para administradores)
- Permite configurar quais dados serão exibidos em cada card através de consultas SQL dinâmicas.
- Customização de cores e tipos de gráficos diretamente pela interface.

---
**Dica:** Utilize os ícones na barra lateral para navegar entre os módulos. O ícone de engrenagem permite ajustar o intervalo de atualização e o modo de operação do sistema.
