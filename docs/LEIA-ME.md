# Documentação do HUB BI

Bem-vindo ao diretório de documentação da plataforma. Aqui você encontrará detalhes sobre como configurar, utilizar e entender a arquitetura do sistema.

## 📁 Arquivos Disponíveis

1.  **[LEIA-ME.md](LEIA-ME.md)**: Este arquivo (Visão Geral).
2.  **[ARQUITETURA.md](ARQUITETURA.md)**: Detalhes técnicos sobre o Backend (FastAPI) e Frontend (Next.js).
3.  **[INSTALACAO.md](INSTALACAO.md)**: Guia passo a passo para configurar o ambiente.
4.  **[GUIA_DO_USUARIO.md](GUIA_DO_USUARIO.md)**: Manual de uso das funcionalidades da plataforma.

## 🚀 Tecnologias Atuais
- **Backend**: FastAPI (Python)
- **Frontend**: Next.js 15 (React)
- **Estilização**: Tailwind CSS
- **Banco de Dados**: SQL Server / SQLAlchemy
- **Visualização**: Recharts / Plotly
